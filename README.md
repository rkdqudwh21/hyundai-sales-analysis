# Hyundai Sales Analysis

현대 자동차 판매 데이터를 기반으로 지역별/월별 판매 흐름을 분석하고,
추천 모델을 도출하는 미니 프로젝트입니다.

## 주요 기능

- 지역별 판매 데이터 분석
- 월별 판매 추이 시각화
- 판매 데이터 기반 추천 모델 도출

## 기술 스택

- Python
- Pandas
- Matplotlib
